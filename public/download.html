📋 Setup Guide — CDJ / Turntable to Broadcast

Mac Setup (with BlackHole)

  1. Download and install BlackHole 2ch from the link above
  2. Open Audio MIDI Setup (built into Mac) → Create a Multi-Output Device combining your speakers + BlackHole
  3. Connect your CDJ/mixer's output to your audio interface (USB)
  4. Open Subversive Radio Broadcaster → scroll to External Input section
  5. Select your audio interface (or BlackHole) from the Device dropdown
  6. Click CONNECT → you'll see the VU meter respond to audio
  7. Hit GO LIVE — your CDJ audio is now broadcasting!

Windows Setup (with VB-Cable)

  1. Download and install VB-Cable from the link above
  2. In Windows Sound Settings → set your CDJ/mixer software output to CABLE Input (VB-Audio)
  3. Open Subversive Radio Broadcaster → scroll to External Input
  4. Select CABLE Output (VB-Audio) from the Device dropdown
  5. Click CONNECT → VU meter should show audio coming in
  6. Hit GO LIVE — broadcasting your external audio!

Another Computer → This Computer

  1. On the sending computer: connect a 3.5mm or USB audio cable from its headphone/line-out to your audio interface's input
  2. Alternatively, both computers on same network? Use NDI Tools (free) or JACK NetDriver for network audio
  3. On this computer: open the Broadcaster → External Input
  4. Select the audio interface receiving the other computer's audio
  5. Click CONNECT and GO LIVE

Direct Browser Capture (Easiest)

  1. Open the Broadcaster in your browser
  2. In the Music Player section, click the 🎵 CAPTURE APP tab
  3. Click "CAPTURE SPOTIFY / YOUTUBE / ANY APP"
  4. Select the tab or app playing music → check "Share audio"
  5. Audio flows directly into your broadcast — digital quality, no cables needed!
